# CineSuggest

## To run the project

## Backend

### First go to the backend and create a virtual environemnt using

python -m venv ./venv

### Then activate the virtual environment

For windows ".\venv\Scripts\activate.bat

### Now type

pip install flask

### Then type to run the app

python server.py

## Frontend

### Now setting up Front end just type

npm i

## Folowed by

npm start
